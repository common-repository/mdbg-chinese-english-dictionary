== Description ==

Automatically links Chinese characters to the [MDBG Chinese-English dictionary](http://www.mdbg.net/chindict/chindict.php), allows conversion of pinyin tone numbers to tone marks and linking pinyin to pronunciation examples.

Please [contact us](http://www.mdbg.net/chindict/chindict.php?page=contact) if you installed this plugin in your blog so we can add it to the list on the [MDBG website](http://www.mdbg.net/chindict/chindict.php?page=wordpress).

== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory, or install it through the admin interface
2. DONE!
